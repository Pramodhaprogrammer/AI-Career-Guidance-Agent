AI Career Guidance Agent

Overview

AI Career Guidance Agent is an AI-powered assistant that helps students identify skill gaps, discover suitable career paths, generate learning roadmaps, and prepare for interviews.

Features

- Skill Gap Analysis
- Career Role Recommendation
- Learning Roadmap Generation
- Interview Preparation Tips
- Project Recommendations

Tech Stack

- Python
- LangChain
- Ollama
- Qwen 2.5

Example

Input

I know Python, SQL and Power BI. Which career suits me?

Output

Recommended Role: Data Scientist

Missing Skills:

- Statistics
- Machine Learning
- Deep Learning

Run the Project

python Agent.py
